Описание API программы

Добавление событий

Для добавления информации о событиях на площадках нужно использовать HTTP GET запрос:

http://[garland_server]/add_event/[address_event_id]/[event_type]/[service_secret]/

В квадратных скобках указаны передаваемые аргументы:

[garland_server] доменное имя или IP адрес программы Garland

[event_type] тип события up или down

[address_event_id] идентификатор адреса из справочника адресов

[service_secret] пин код службы мониторинга, для идентификации службы мониторинга
